# 🎬 Tawsiya (توصية) - AI Movie Recommendation App
An intelligent, modern, and responsive movie recommendation web application built with React and Vite. It utilizes the powerful TMDB API combined with custom algorithmic scoring to deliver highly accurate, personalized movie suggestions tailored to the user's taste.

## ✨ Features
- **🤖 AI-Powered Recommendations:** Search for any movie and receive intelligent suggestions. Our custom algorithm ranks recommendations based on exact title similarity, shared genres, semantic keyword matching in the plot/description, and global popularity.
- **🔍 Smart Search & Discovery:** A robust search engine allowing users to filter movies by genre, release year, minimum rating, and watch status. Includes debounced text search and a dedicated "Discover" mode.
- **🌍 Bilingual Support:** Seamlessly switch between Arabic (Default, RTL) and English (LTR) interfaces throughout the entire application.
- **📱 Responsive & Cinematic Design:** A premium, glassmorphism-inspired UI with smooth Framer Motion animations, optimized for desktop, tablet, and mobile devices.
- **👥 Actor & Director Filmography:** Explore the entire catalog of movies directed by or starring your favorite cinema personalities.
- **👤 User Profiles & Libraries:** Users can manage their "Watchlist" and "Favorites", mark movies as watched, and customize their profile avatars with culturally relevant designs.
- **⚡ Fast & Modern Setup:** Built on Vite for lightning-fast Hot Module Replacement (HMR) and optimized production builds.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Custom CSS, CSS Variables, Glassmorphism UI
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Routing:** React Router DOM
- **Data Source:** TMDB (The Movie Database) API

## 🚀 Quick Start

### Requirements
- Python 3.8 or higher (If using backend automation)
- Node.js installed
- A free API key from [TMDB](https://www.themoviedb.org/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root of the project and add your TMDB API Key:
   ```env
   VITE_TMDB_API_KEY=your_tmdb_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to view the app!

## 📦 Building for Production

To create an optimized production build (e.g., for deployment on Vercel, Netlify, or GitHub Pages), run:
```bash
npm run build
```
This will generate a `dist` folder containing all the optimized static files.
